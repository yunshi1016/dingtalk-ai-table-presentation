# 钉钉 AI 表格业务场景共创方案

面向老板与管理层的 20 分钟以内现场演示网页。方案以业务价值为主线，产品素材仅用于支撑关键能力、真实场景和成果验收。

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开终端输出的本地地址即可演示。

## 演示操作

- `右方向键`、`空格`、`PageDown`：下一页
- `左方向键`、`PageUp`：上一页
- `Home`、`End`：首尾页
- `N`：打开或关闭演讲备注
- `O`：打开或关闭页面目录
- 页面底部按钮：翻页、目录、备注和全屏

手机端支持左右滑动翻页。

## 构建

```bash
npm run build
```

构建结果输出到 `dist` 目录。
